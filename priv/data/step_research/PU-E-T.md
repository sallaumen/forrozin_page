# PU-E-T - Push n pull com esquerda para tras

Variacao do push n pull que acontece pos-dissociacao de abertura. O condutor executa o empurra-puxa, mas a resolucao vem com o pe esquerdo indo para tras em vez do padrao frontal. Isso exige dissociacao: mover tronco e quadril em direcoes diferentes, pois o tronco conduz para frente enquanto o quadril resolve para tras.

E uma variacao que demonstra dominio da dissociacao, conceito tecnico essencial para giros de 5 pisadas e inversoes.
